# HuggingChat API Examples

## Installation

### Windows

```{bash}
python -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
```

### Linux

```{bash}
python -m venv .venv
.venv/bin/activate
pip install -r requirements.txt
```

## Usage

### CLI

```{bash}
python cli.py
```
